Question 1: Which CSS properties each paragraph receives

Paragraph A:
  Matches only the element selector "p".
  Gets: color: gray, font-size: 1rem, font-family: Georgia serif, line-height: 1.4, padding: 4px 0.

Paragraph B:
  Matches "p" (element) and ".info" (class).
  Class has higher specificity than element, so .info wins where they overlap.
  Gets from .info: color: steelblue, font-size: 1.25rem, font-family: Arial sans-serif, background-color: #e8f4fd, padding: 8px 14px, border-left: 5px solid steelblue.
  Gets from p (not overridden): line-height: 1.4.

Paragraph C:
  Matches "p" (element) and "#warning" (id).
  ID has higher specificity than element, so #warning wins where they overlap.
  Gets from #warning: color: darkred, font-size: 0.8rem, font-style: italic, text-transform: uppercase, letter-spacing: 0.15em, border: 2px dashed darkred, padding: 10px, background-color: #fff0f0, font-family: Courier New monospace.
  Gets from p (not overridden): line-height: 1.4.

Paragraph D:
  Matches "p" (element), ".info" (class), "#warning" (id), and "#warning.info" (id+class).
  Specificity order: p (0,0,1) < .info (0,1,0) < #warning (1,0,0) < #warning.info (1,1,0).
  #warning.info has the highest specificity and wins where it defines a property.
  Gets from #warning.info: color: darkorange, background-color: #fff7e0, border: 2px solid darkorange, font-size: 1.1rem, font-style: normal, letter-spacing: 0.03em.
  Gets from #warning (not overridden by #warning.info): text-transform: uppercase, font-family: Courier New monospace.
  Gets from p (not overridden): line-height: 1.4.

Question 2: Color and font-family for Paragraph D

Color: all four rules define color. The browser uses darkorange from #warning.info because it has the highest specificity (1,1,0 = id + class). Specificity order: p gray (0,0,1) < .info steelblue (0,1,0) < #warning darkred (1,0,0) < #warning.info darkorange (1,1,0).

Font-family: only "p" (Georgia) and ".info" (Arial) and "#warning" (Courier New) define font-family. #warning.info does not define font-family. The highest specificity that defines font-family is #warning (1,0,0), so the browser uses Courier New monospace.

Question 3: External vs internal CSS for h1

The external style.css defines h1 with color: purple, font-style: italic, padding: 10px.
The internal <style> in the <head> also defines h1 with color: green.
Both have the same specificity (element selector = 0,0,1). When specificity is equal, the rule that appears last in source order wins. The internal <style> comes after the <link> to style.css, so the internal color: green overrides the external color: purple. The font-style and padding from style.css still apply because the internal rule does not redefine them.
